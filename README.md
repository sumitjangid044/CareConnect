✏️    # CareConnect 🏥

CareConnect is a Doctor Appointment Booking Platform built using the MERN Stack.

## 🚀 Live Demo

Frontend:
https://your-frontend.vercel.app

Admin Panel:
https://your-admin.vercel.app

Backend API:
https://careconnect-backend-1-zypb.onrender.com

## ⚙️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- Cloudinary
- Stripe Payment

## ✨ Features

- User Authentication
- Doctor Listing
- Appointment Booking
- Admin Dashboard
- Doctor Management
- Payment Integration

## 📂 Project Structure
Frontend : User website
admin → Admin dashboard
Backend → API server
